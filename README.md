## Run Locally

Clone the project

```bash
  git clone https://github.com/LUIS9799/MusicBot.git
```

Go to the project directory

```bash
  cd MusicBot
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index.js
```
